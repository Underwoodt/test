## Quick Fixes

### Services Won't Start
Check logs: docker compose logs

### Can't Access UI
Check: http://frontend.localhost

### API Not Responding
Check: curl http://localhost:8086/health

### Database Issues
Check connectivity to MongoDB and PostgreSQL
